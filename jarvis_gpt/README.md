# Jarvis-GPT Mil Grau

## Instalação

Para instalar tudo que é necessário para rodar
```
python -m pip install -r requirements.txt
```

No Arduino é preciso instalar a biblioteca Firmata e gravar no Arduino o exemplo StandardFirmata
